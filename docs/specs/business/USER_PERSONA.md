# Persona do Usuário — ATLAS

## Perfil: Enzo

### Demografia
- Desenvolvedor de software
- Único usuário do sistema
- Familiarizado com tecnologia e APIs
- Usa Obsidian como "segundo cérebro"
- Smartphone Android como device principal

### Objetivos
1. **Visão consolidada do dia** — Saber o que tem pra fazer, compromissos e estado dos hábitos num único lugar
2. **Captura sem fricção** — Registrar notas, eventos e hábitos via linguagem natural sem abrir múltiplos apps
3. **Consistência de hábitos** — Manter tracking regular de sono, exercício e humor
4. **Interação divertida** — Ter um assistente com personalidade que torne o uso prazeroso

### Pain Points (sem ATLAS)
| Pain Point | Severidade | Contexto |
|---|---|---|
| Falta de visão geral do dia | Alta | Informações espalhadas entre Calendar, Obsidian, apps de hábitos |
| Fricção para anotar | Média | Abrir app → navegar → criar nota → organizar = muitos passos |
| Assistentes de voz genéricos | Média | Siri/Google Assistant não têm personalidade nem contexto pessoal |
| Esquecimento de hábitos | Média | Sem prompt proativo, esquece de registrar |

### Comportamento de Uso
- **Frequência esperada:** Uso diário, múltiplas vezes ao dia
- **Horários pico:** Manhã (briefing), ao longo do dia (captura), noite (review + hábitos)
- **Modalidade:** Texto + voz via app mobile
- **Tolerância a erros:** Baixa para classificação errada de intenção; alta para sarcasmo

### Preferências de Comunicação
- Respostas diretas e concisas
- Tom sarcástico é o diferencial — o que faz querer usar
- Sem emojis, sem formalidade corporativa
- Observações proativas sobre padrões são bem-vindas
- Português brasileiro nativo

### Contexto Tecnológico
- Confortável com APIs, terminal, configuração técnica
- Usa Obsidian ativamente como vault de conhecimento
- Google Calendar como agenda principal
- Android como plataforma mobile
