# Panorama Competitivo — ATLAS

## Contexto

ATLAS não compete num mercado — é um projeto pessoal. Mas substitui ferramentas existentes na rotina do usuário. Esta análise mapeia o que o ATLAS substitui e por quê.

## Ferramentas Substituídas

### Google Calendar (direto)
**O que faz:** Gerenciamento de agenda
**Por que substituir:** Interface requer múltiplos taps para criar/ver eventos. ATLAS permite via linguagem natural em uma mensagem.
**Vantagem ATLAS:** Captura por voz/texto + personalidade + integração com briefing
**Limitação ATLAS:** Menos controle granular (sem edição de eventos recorrentes, sem convites)

### Apps de Hábitos (Habitica, Streaks, etc.)
**O que fazem:** Tracking de hábitos com gamificação ou streaks
**Por que substituir:** App separado = mais um app para abrir. Dados isolados.
**Vantagem ATLAS:** Registro em linguagem natural + dados integrados no vault + briefing
**Limitação ATLAS:** Sem visualização gráfica, sem gamificação, sem lembretes (MVP)

### Assistentes de Voz (Siri, Google Assistant)
**O que fazem:** Comandos por voz genéricos
**Por que substituir:** Sem contexto pessoal, sem personalidade, sem integração com vault
**Vantagem ATLAS:** Personalidade sarcástica, acesso ao vault, contexto acumulado
**Limitação ATLAS:** Menos integrações (sem controle de casa, sem música, sem apps terceiros)

### Obsidian (direto)
**O que faz:** Editor de notas markdown
**Não substitui:** ATLAS complementa — cria notas via linguagem natural, mas Obsidian continua sendo usado para edição e navegação manual.
**Relação:** Simbiótica, não competitiva

## Diferencial Insubstituível

A combinação de **personalidade + contexto pessoal + captura sem fricção** não existe em nenhuma ferramenta disponível. Cada componente sozinho tem alternativas melhores, mas a integração é única.
