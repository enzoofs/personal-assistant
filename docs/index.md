# Documentação — ATLAS

## Visão Geral

ATLAS é um assistente pessoal inteligente com personalidade sarcástica (estilo Jarvis) que centraliza informações em um vault Obsidian, gerencia agenda via Google Calendar, faz busca inteligente e rastreia hábitos. Projeto pessoal, single-user.
